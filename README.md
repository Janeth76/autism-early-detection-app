# Early Autism Detection Using AI

This project uses Machine Learning in Python to identify early behavioral indicators of Autism Spectrum Disorder (ASD) in children between 5 months and 5 years old.

The model analyzes behavioral indicators such as:

- Eye contact
- Response to name
- Social interaction
- Language delay
- Repetitive movements

Technology used:
- Python
- Scikit-learn
- Machine Learning (Random Forest)

Purpose:
To support early screening and help professionals identify children who may benefit from early intervention.
